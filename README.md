# Lacanau Volley-Ball — Site web

Site moderne du club de volley loisir de Lacanau (Next.js, Tailwind CSS).

## Lancer le projet

```bash
npm install
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000).

## Build production

```bash
npm run build
npm start
```

## Stack

- **Next.js 16** (App Router)
- **React 19**
- **Tailwind CSS 4**
- **TypeScript**
- **lucide-react** (icônes)

## Structure

- `src/app/` — pages et layout
- `src/components/` — Header (TopBar, Nav, MobileMenu), Home (Hero, News, NextMatch, CTAs, Partners, Social), Footer
- Pages : accueil, le-club, pratique, tarifs, calendrier, resultats, partenaires, contact, billetterie, boutique, infos
